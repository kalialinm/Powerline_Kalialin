## Overview

This repository contains a Python script for analyzing NEM bid-price offers and price setting behaviou, including:

- Processing bid data (`raw_bid_data_2025-06-26.csv`).
- Reconstructuon supply curves.
- Plotting supply curves per interval and region.
- Identifying price-setting generators and patterns.
- Counterfactual analysis (e.g., adding hypothetical wind bids).
- BESS-related analysis.
